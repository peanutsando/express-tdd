### Express TDD Example

#### Express usage Quick Start

```
$ npm install -g express-generator@4
$ express /express-tdd && cd /express-tdd
$ npm install
$ npm start
```

#### Test Script

```
$ npm run test:index
$ npm run test:user
```

#### TDD Package

```
$ npm install mocha
$ npm install supertest
$ npm install chai
```
